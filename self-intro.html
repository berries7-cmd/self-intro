사용자가 사랑하는 경험을 설계하는 개발자.

컴퓨터공학 전공 · 스타트업 2년 · 문제 해결에 집착하는 풀스택 메이커 김민수입니다.

지원 동기 & 강점

혁신적인 개발 문화를 가진 팀에서 사용자 중심 가치를 극대화하고, 빠른 실험과 반복으로 의미 있는 제품 임팩트를 만드는 것이 목표입니다.

저를 움직이는 이유

사용자 경험을 개선하는 솔루션을 찾을 때 가장 큰 에너지를 얻습니다. 귀사의 제품에 저의 UX 감각과 실행력을 더해 실제 고객이 체감하는 변화를 만들고 싶습니다.

제가 제공하는 가치

  • 끊임없는 학습으로 최신 기술을 빠르게 흡수
  • 직군 간 언어를 번역하는 협업형 커뮤니케이션
  • 문제를 끝까지 추적하며 해결하는 집요함
  • 완벽주의를 통제하는 우선순위 기반 실행력

철학

“기술은 사람을 위해 존재한다.”

생활 신조

화려한 기술보다 사용자가 실제로 편리함을 느끼는지에 집중합니다. 문제를 사용자 관점에서 다시 정의하고, 효율적인 업무 경험으로 연결될 때 비로소 기술이 의미 있다고 믿습니다.

협업 방식

디자이너, 기획자, 동료 개발자와 목표를 명확히 나누고 실행 과정을 투명하게 공유합니다. 후배 멘토링 경험으로 복잡한 개념을 쉽게 설명하는 능력 역시 갖추고 있습니다.

주요 경험

스타트업 웹 개발자 · 2년

사용자 피드백을 기반으로 한 빠른 개선 사이클

프론트와 백엔드를 넘나들며 웹 서비스를 운영했습니다. 반복되는 실험과 지표 분석을 통해 사용자 경험을 개선하고 비즈니스 목표를 동시 달성했습니다.

지역 상권 활성화 앱 · 해커톤 우수상

48시간 안에 React Native 풀스택 완성

팀장을 맡아 API 설계부터 UI까지 전 과정을 주도했습니다. 짧은 시간 내에도 팀원의 역량을 읽고 역할을 분배하는 리더십을 증명했습니다.

컴퓨터공학 학회 멘토 · 4년

복잡한 개념을 쉽게 설명하는 커뮤니케이션

학회에서 후배들에게 프로그래밍을 가르치며 지식을 구조화하는 능력을 키웠고, 이는 현재 코드 리뷰와 기술 공유 문화에 그대로 적용하고 있습니다.

입사 후 로드맵

단기 · Alignment

팀 문화와 도메인을 빠르게 흡수하고, 맡은 업무를 안정적으로 완수해 신뢰를 쌓습니다. 코드 컨벤션과 협업 도구에 즉시 동화되어 생산성을 끌어올립니다.

중기 · Elevation

사용자 중심 기능을 고도화하고 개발 프로세스를 개선합니다. 지표 중심의 의사결정으로 서비스 품질을 높이며, 팀 내 지식 공유를 주도합니다.

장기 · Leadership

클라우드·마이크로서비스 전문성을 확장하고 기술 리더로 성장합니다. 주니어 개발자를 멘토링하며, 새로운 기술 도입과 조직의 기술 방향성을 리드하겠습니다.
